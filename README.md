# fintech-data-pipeline

> Work in progress — actively building

## Project Overview
An end-to-end automated financial data pipeline featuring 
PostgreSQL, Python ETL, Power BI dashboards, and a 
Google Gemini AI fraud detection agent.

## Tech Stack
- Database: SQLite (dev) → PostgreSQL (production)
- ETL: Python, Pandas, SQLAlchemy
- Visualization: Power BI
- AI Integration: Google Gemini API

## Status
- [x] Phase 1 — Database design & initial data load
- [ ] Phase 2 — Automated Python ETL pipeline
- [ ] Phase 3 — SQL performance tuning
- [ ] Phase 4 — Power BI dashboard
- [ ] Phase 5 — AI audit agent
- [ ] Phase 6 — Portfolio deployment

PaySim synthetic financial fraud dataset — 6.3M transactions
